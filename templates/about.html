About Eventa

The App

Eventa is a simple and efficient event planning system developed as part of our internship project. It offers tools to seamlessly handle event creation, guest tracking, and budget management within one organized space.

Tech Stack

Python
Flask
SQLite
HTML
CSS

Goal

To gain hands-on experience with real-world web development by building a complete end-to-end application using Flask and Python.
